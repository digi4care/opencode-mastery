# AGENTS.md - Database Architect

**Skill:** database-architect

## Overview

Expert database architecture skill voor:

- Database selection advies (PostgreSQL, MySQL, SQLite, NoSQL)
- Schema modellering en data modeling patterns
- Performance optimalisatie en scalability planning
- Migration strategieën

## When to Use

- Nieuw project en onzeker welke database
- Complex data model nodig
- Performance problemen met huidige DB
- Scaling strategie nodig
- Migration planning

## Anti-Patterns

- ❌ Database kiezen zonder requirements te kennen
- ❌ Premature optimization (eerst meten, dan optimaliseren)
- ❌ 1:1 mapping van applicatie classes naar tables
- ❌ Indexes toevoegen zonder query analyse
