# OpenCode Examples

Production-ready examples for OpenCode skills, plugins, and tools.

## Structure

```
examples/
├── skills/           # Skill examples
│   ├── minimal/      # Single SKILL.md
│   ├── intermediate/ # SKILL.md + helper script
│   └── advanced/     # Full workflow with config + scripts
├── plugins/          # Plugin examples
│   ├── minimal/      # Event logging
│   ├── intermediate/ # Custom tools + agents
│   └── advanced/     # Full capabilities
└── tools/            # Tool examples
    ├── minimal/      # Single tool
    ├── intermediate/ # Multiple tools
    └── advanced/     # Multi-tool with utils
```

## Tiers

| Tier             | Level  | Description                     |
| ---------------- | ------ | ------------------------------- |
| **minimal**      | \*     | Simplest working example        |
| **intermediate** | \*\*   | Common patterns, validation     |
| **advanced**     | \*\*\* | Production-ready, full features |

3. **Advanced**: `skills/advanced/` - Full workflow orchestration
