## Code Examples

### Example 1

```jsx
<script lang="ts">
  import { Slider } from "$lib/components/ui/slider/index.js";

  let value = $state(50);
</script>

<Slider type="single" bind:value max={100} step={1} class="max-w-[70%]" />
```
