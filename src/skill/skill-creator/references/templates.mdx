# Templates

Use these templates for consistent tool inputs.

## Plan Input

```json
{
  "request": "Create a skill for incident postmortems",
  "triggers": ["incident postmortem", "summarize incident report"],
  "workflow": ["Collect report", "Extract timeline", "Write summary"],
  "errorHandling": ["If report is missing, ask for report path"],
  "tests": {
    "shouldTrigger": ["Summarize this incident report"],
    "shouldNotTrigger": ["Explain React hooks"],
    "functional": ["Draft a postmortem from this incident report"]
  },
  "references": ["workflow", "checklist"]
}
```

## Create Input

```json
{
  "request": "Create a skill for incident postmortems",
  "baseDir": "src/skill",
  "dryRun": true,
  "confirm": false
}
```
