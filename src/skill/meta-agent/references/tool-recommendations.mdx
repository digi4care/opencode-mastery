# Tool Recommendations

Use this matrix to choose the right component for the job.

| Need                        | Best Choice | Why                         |
| --------------------------- | ----------- | --------------------------- |
| Explicit, manual workflow   | Command     | User invokes on demand      |
| Automatic guidance          | Skill       | Auto-triggered instructions |
| Custom tools or hooks       | Plugin      | Full SDK and event hooks    |
| Dedicated assistant         | Agent       | Scoped tools + instructions |
| External system integration | MCP         | Cross-system capability     |

## Clarification Questions

- Do you want this to run only when you explicitly call it?
- Should this auto-trigger based on user input?
- Do you need custom tools or event hooks?
- Should a dedicated agent handle this workflow?
- Do we need to call an external service?
