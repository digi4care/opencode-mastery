# AGENTS.md - Svelte MCP Skill

**Skill:** Svelte MCP Server - AI-assisted Svelte/SvelteKit development

## Overview

Model Context Protocol server voor Svelte: documentatie lookup, code analyse, en playground integratie.

## Capabilities

| Feature       | Description                   |
| ------------- | ----------------------------- |
| Documentation | Real-time Svelte docs lookup  |
| Code Analysis | svelte-autofixer suggestions  |
| Playground    | Generate shareable code links |

## Tools

### svelte-autofixer

Automatische fixes voor:

- Syntax errors
- Best practices
- Performance optimalisaties
- Accessibility (a11y)

### Documentation Search

Topics beschikbaar:

- Components, props, slots
- Reactivity (runes, stores)
- Directives (bind, on, use)
- SvelteKit routing

## When to Use

- Svelte documentatie opzoeken
- Code review en analyse
- Best practices vragen
- Debugging hulp
- Playground links genereren
